1. Introduction

Happymoves is a virtual reality–based rehabilitation and movement training platform developed and maintained by Oqulix Private Limited. This Privacy Policy explains what information we collect when you use Happymoves (VR application and connected web application), how we use it, how we protect it, and your choices.

2. Information We Collect

From the VR application

When you use Happymoves on Meta Quest or other VR devices, we collect only the minimal data required for core functionality, licensing, and analytics:

  • Device ID: Used to verify license activation and associate session records with the right device.
  • Session data: Session duration, performance scores, counts/repetitions, difficulty settings, and gameplay settings needed to track rehabilitation progress.
  • Optional movement data: Hand position or motion‑tracking data - collected only if the player enables motion tracking in player settings.
  • Diagnostic data: Crash reports and diagnostics used to improve app stability and performance.

Important: The VR application does not collect personal contact details such as names, email addresses, phone numbers, or payment information from the device itself.

From the Web Platform

When users or therapists register through the Happymoves web platform we may collect the following information and store it securely in Firebase Firestore:

  • Player name or identifier: A name or identifier entered by a user or therapist to label sessions and profiles.
  • Associated device ID and session records: Links VR session data to a specific player profile so therapists can review progress.

3. How We Use the Information

We use collected data strictly for the following purposes:

  • Enable license activation and authenticate devices.
  • Track and display session progress (duration, repetitions, performance scores).
  • Provide analytics and insights to therapists or administrators managing rehabilitation programs.
  • Improve app performance, stability, feature development, and user experience.

We do not sell, rent, or share identifiable user data with third parties for marketing purposes.

4. Data Storage & Security

All user and session data for Happymoves is stored in Firebase Firestore (Google Cloud). Google maintains compliance programs and certifications such as GDPR, CCPA controls, and ISO/IEC 27001.

Key security measures:

  • Data in transit is encrypted using HTTPS/TLS.
  • Access to production databases is restricted by role-based authentication and least-privilege access controls.
  • Session data is retained for rehabilitation tracking. Users may request deletion (see below).

Note: Oqulix Private Limited itself is not separately certified under these standards; we rely on the security and compliance of our cloud providers (e.g., Firebase).

5. User Data Deletion Requests

Users may request deletion of all data associated with their name, player ID, or device ID at any time. To request deletion, contact our privacy team at:

contact@oqulix.com

Upon verification of the request, we will permanently delete related records from Firebase within 15 business days. We may retain limited anonymized or aggregated data for product analytics or legal compliance where required.

6. Third‑Party Services

Happymoves integrates with a small set of third‑party services for core functionality:

  • Firebase (Google LLC): Secure data storage, analytics, and authentication.
  • Unity Analytics: Non-identifiable usage metrics and crash reporting for improving stability.

These services process only the limited data required for their roles and do not receive personally identifiable information from the VR application by default.
